TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

Problem: "Backend won't start"
Solution: 
  1. Check .env for BREVO_API_KEY: grep BREVO_API_KEY backend/.env
  2. Reinstall: python3 -m pip install -r backend/requirements.txt
  3. Check logs: tail -f /tmp/blinkmail_backend.log

Problem: "Emails not sending"
Solution:
  1. Verify Brevo API key is valid
  2. Check logs for error message
  3. Ensure recipient email is valid format

Problem: "Rate limited"
Solution:
  1. Free tier: 300/day limit - upgrade if needed
  2. Wait before retry
  3. Upgrade Brevo account for higher limits

Problem: "Module not found: email_provider"
Solution:
  1. Verify file exists: ls backend/email_provider.py
  2. Check Python path: cd backend && python3 -c "import email_provider"
  3. Reinstall if needed
